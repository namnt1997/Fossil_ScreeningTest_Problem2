# My Alarm
## Solve problem 2 of Fossil Vietnam's Android screening test

Build a simple Alarm application which provides following features:
1. Create new alarm, user can pick time, update time, select repeat option.
2. View created alarm list, can enable/disable each alarm.
3. When alarm goes off, the application must fire a notification with sound and vibration for user.
4. The application should be able to run in background.
